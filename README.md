## Manual Testing

Prepared manual test cases for both the Login and Signup modules.

### Login Test Cases

- Positive test scenarios
- Negative test scenarios
- Field validations
- UI validations
- Functional validations

### Signup Test Cases

- Required field validations
- Password validation
- Confirm password validation
- Mobile number validation
- Terms & Conditions validation
- UI and functional test scenarios

Each test case includes:

- Test Case ID
- Test Case Name
- Preconditions
- Test Steps
- Expected Result
- Actual Result
- Status
- Remarks
